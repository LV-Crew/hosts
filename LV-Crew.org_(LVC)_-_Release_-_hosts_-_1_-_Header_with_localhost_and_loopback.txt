# This is a HOSTS file used by (Microsoft) TCP/IP (for Windows).
#
# This file contains the mappings of IP addresses (IP4/IP6) to host names.
# Each entry should be kept on an individual line. The IP address should
# be placed in the first column followed by the corresponding host name.
# The IP address and the host name should be separated by at least one
# space.
#
# Additionally, comments (such as these) may be inserted on individual
# lines or following the machine name denoted by a '#' symbol.
#
# For example:
#      102.54.94.97     rhino.acme.com          # source server
#       38.25.63.10     x.acme.com              # x client host
#
# localhost name resolution is handled within DNS itself.
#	127.0.0.1       localhost
#	::1             localhost
#
# Modified default host file header from Microsoft by LVC
# Source: %SystemRoot%\System32\drivers\etc\


# You can add the following list to your host file, or replace your excisting one.


# Here follows a list of the locations of the host file in the file system depending on the OSes.
#
# The location of the hosts file in the file system hierarchy varies by operating system.
# It is usually named hosts, without an extension.
#
# Operating System: 		Version(s): 		Location:
# Unix, Unix-like, POSIX 				/etc/hosts[3]
# Microsoft Windows 		3.1 			%WinDir%\HOSTS
# 				95, 98, ME 		%WinDir%\hosts
# 				NT, 2000, XP, 2003, Vista, 2008, 7, 2012, 8, 10 
# 							%SystemRoot%\System32\drivers\etc\hosts
# 							E.g.: C:\Windows\System32\drivers\etc\hosts
# Windows Mobile, Windows Phone 			Registry key under HKEY_LOCAL_MACHINE\Comm\Tcpip\Hosts
# Apple Macintosh 		9 and earlier 		Preferences or System folder
# Mac OS X 			10.0–10.1.5 		(Added through NetInfo or niload)
# 	 			10.2 and newer		/etc/hosts (a symbolic link to /private/etc/hosts)
# Novell NetWare 					SYS:etc\hosts
# OS/2 & eComStation 					"bootdrive":\mptn\etc\
# Symbian OS 			6.1–9.0 		C:\system\data\hosts
# 				9.1+ 			C:\private\10000882\hosts
# MorphOS 			NetStack 		ENVARC:sys/net/hosts
# AmigaOS 			4	 		DEVS:Internet/hosts
# AROS				 			ENVARC:AROSTCP/db/hosts
# Android 						/etc/hosts (a symbolic link to /system/etc/hosts)
# iOS 				2.0 and newer 		/etc/hosts (a symbolic link to /private/etc/hosts)
# TOPS-20 						<SYSTEM>HOSTS.TXT
# Plan 9 						/lib/ndb/hosts
# BeOS 							/boot/beos/etc/hosts
# Haiku 						/boot/common/settings/network/hosts
# OpenVMS 			UCX 			UCX$HOST
# 				TCPware 		TCPIP$HOST
